# Cybersecurity Awareness Chatbot - PROG6221 POE Part 3

## Features
- ✅ Task Assistant with MySQL Database (Add, View, Reminders, Mark Complete, Delete)
- ✅ Interactive Cybersecurity Quiz (12 questions with feedback)
- ✅ NLP Simulation (Keyword detection for flexible input)
- ✅ Activity Log with timestamps
- ✅ Chatty & Friendly Personality (remembers user name)
- ✅ Full WPF GUI

## How to Run
1. Open in Visual Studio
2. Install `MySql.Data` NuGet package
3. Update ConnectionString with your MySQL password
4. Run the application

## Technologies
- C# / WPF
- MySQL Database
- MySql.Data Connector
